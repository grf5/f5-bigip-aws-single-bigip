# F5 BIG-IP AWS Transit Gateway Route Table update via Lambda (HA)

## Overview

Terraform plan to deploy a POC of BIG-IP HA with transit gateway and Lambda-updated routing

## Getting Started

1. Copy the admin.tfvars file and update the variables.
2. Review and edit the variables.tf file, where/if necessary.
3. Run the setup.sh script.
